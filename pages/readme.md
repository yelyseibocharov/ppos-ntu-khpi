# PAGES

## Ця папка призначена для структуризації сторінок і компонентів сайту.

### Якщо ви створюєте якусь сторінку, створюєте папку з її назвою(призначенням) і заповнюєте там html i css файли.

### Якщо ж компонент, створюєте його всередині папки components (або створіть її якщо не існує)

#### Наприклад:

`pages/components/header`
`pages/news/...`
